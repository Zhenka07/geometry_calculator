## circle.py
>* `area(r: (float/int)) --> (float/int)` - Функция принимает на вход радиус круга и возвращает его оплощадь.
>```python
>from circle import area
>area(5) # return: 49.3
>```
* `perimetr(r: (float/int)) --> (float/int)` - Функция принемает на вход радиус круга и возвращает его периметр.
>```python
>from circle import area
>area(5) #return: 31.4
>```

## square.py
>+ `area(r: (float|int))->(float|int)` - Функция принимает на вход сторону квадрата и возвращает его площадь. 
>```python
>from square import area
>area(5) #return: 25
>```
+ `perimeter(r: (float|int))->(float|int)` - Функция принимает на вход сторону квадрата и возвращает его периметр. 
>```python
>from square import area
>area(5) #return: 20
>```
