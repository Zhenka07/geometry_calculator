Библиотека имеет модули для вычисления площaди и периметра круга и квадрата
---
# <font color="red">Функции</font>
## circle.py
>* `area(r: (float/int)) --> (float/int)` - Функция принимает на вход радиус круга и возвращает его оплощадь.
>```python
>from circle import area
>area(5) # return: 49.3
>```
* `perimetr(r: (float/int)) --> (float/int)` - Функция принемает на вход радиус круга и возвращает его периметр.
>```python
>from circle import area
>area(5) #return: 31.4
>```
